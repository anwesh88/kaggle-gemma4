Pipeline truths worth preserving

  1. Yahoo is inbound only — it contributes public prices, not user behavior.
  2. Kite is optional — only Live Kite mode opens the broker path.
  3. Behavioral context is local — vows, scores, DNA, and the SEBI corpus stay on-device.
  4. Gemma sees an explicit prompt — TradingContext, vows, history, and live-risk fields.
  5. Gemma returns strict JSON — score, pattern, localized nudge, and vow violations.
  6. SEBI RAG is attached after inference — local disclosure, not hidden prompt input.
  7. Completed runs become future memory — persistence improves the next analysis.
  8. The final product output is action — insight culminates in a pre-order intervention.
