# Gemma Model Variant Naming and Attribution

Finsight OS follows the external Gemma model variant guidelines:

- The app brand is **Finsight OS**, not Gemma.
- External fine-tuned artifacts use distinct Finsight names, such as
  `finsight-guardian-sebi-lora-v1`.
- Gemma is referenced only when accurately describing the base model family or
  runtime, such as "based on Gemma 4 E2B" or `OLLAMA_MODEL=gemma4:e2b`.
- No wording in the project should imply endorsement, sponsorship, or
  affiliation with Google.
- Gemma marks are not used as Finsight OS logos or primary brand assets.

Required attribution:

> Gemma is a trademark of Google LLC.

Project disclaimer:

> Finsight OS is an independent project and is not affiliated with or endorsed by Google.
